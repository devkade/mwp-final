# 제작 목적 및 필요성 (Purpose & Necessity)

## 1. 프로젝트 배경

### 1.1 현재 문제점

헬스장 운영에서 발생하는 주요 문제들:

| 문제 | 현황 | 영향 |
|------|------|------|
| **수동 관리** | 기구 사용량을 수기나 직감으로 파악 | 정확한 데이터 부재 |
| **비효율적 배치** | 기구 배치가 경험에 의존 | 공간 활용도 저하 |
| **유지보수 지연** | 고장 후 대응 | 고객 불만, 수리 비용 증가 |
| **혼잡도 불투명** | 실시간 현황 파악 불가 | 고객 대기 시간 증가 |

### 1.2 해결 목표

**AI 기반 자동화 시스템**으로 위 문제들을 해결:

```
┌─────────────────────────────────────────────────────────────┐
│                    GymFlow 핵심 가치                         │
├─────────────────────────────────────────────────────────────┤
│  🎯 자동 사용량 추적    →   정확한 데이터 수집                │
│  📊 데이터 기반 분석    →   객관적 의사결정                   │
│  ⚡ 실시간 모니터링     →   즉각적 현황 파악                  │
│  🔧 예방적 유지보수     →   비용 절감                         │
└─────────────────────────────────────────────────────────────┘
```

## 3. 필요성

### 3.1 운영 효율성 측면

#### 자동화된 사용량 추적

| AS-IS | TO-BE |
|-------|-------|
| 직원이 직접 확인하거나 추정 | AI가 자동으로 감지 및 기록 |
| 주관적, 부정확한 데이터 | 객관적, 정확한 데이터 |
| 실시간 파악 불가 | 실시간 모니터링 가능 |

#### 데이터 기반 의사결정

축적된 데이터를 통해:
- 인기 기구 / 방치 기구 파악
- 피크 타임 분석
- 기구 재배치 근거 확보
- 신규 기구 도입 판단

### 3.2 비용 절감 측면

#### 유휴 자원 최소화

```
사용률 저조 기구 파악
        ↓
공간 재배치 또는 기구 교체
        ↓
공간 효율 증대 + 고객 만족도 향상
```

#### 예방적 유지보수

| 기존 방식 | GymFlow 적용 후 |
|----------|----------------|
| 고장 후 수리 | 사용량 기반 예측 |
| 돌발 비용 발생 | 계획적 유지보수 |
| 고객 불만 | 서비스 연속성 유지 |

### 3.3 사용자 경험 측면

#### 실시간 상태 확인 (향후 확장)

- 앱으로 기구 사용 현황 확인
- 대기 시간 예측
- 빈 기구 알림

#### 운동 기록 자동화

- 별도 기록 없이 히스토리 확인 가능
- 사용자 식별 연동 시 개인별 통계 제공

### 3.4 보안 및 확장성 측면

#### 24시간 무인 모니터링

| 기능 | 효과 |
|------|------|
| 야간/무인 시간대 모니터링 | 관리자 상주 불필요 |
| 이벤트 이미지 저장 | 증거 자료 확보 |
| 이상 상황 감지 | 오용/파손 사후 확인 |

#### 유연한 연동성

RESTful API 제공으로:
- 회원 관리 시스템(CRM) 연동
- IoT 디바이스 연동
- 타 서비스 확장 용이

## 5. 활용 시나리오

### 5.1 헬스장 관리자

```
1. 웹 대시보드 접속
2. 전체 기구 사용 현황 확인
3. 사용률 저조 기구 식별
4. 재배치 또는 교체 결정
5. 효과 모니터링
```

### 5.2 헬스장 이용객 (향후)

```
1. 모바일 앱 접속
2. 실시간 기구 사용 현황 확인
3. 빈 기구로 이동
4. 대기 시간 최소화
```

### 5.3 유지보수 담당자

```
1. 기구별 사용량 데이터 조회
2. 높은 사용량 기구 식별
3. 예방 정비 스케줄링
4. 고장 전 선제 대응
```

## 6. 결론

GymFlow는 **AI 객체 감지 기술**을 활용하여 헬스장 운영의 핵심 문제들을 해결합니다.

### 핵심 가치

| 가치 | 실현 방법 |
|------|----------|
| **자동화** | YOLOv5 기반 자동 사용 감지 |
| **정확성** | 5초 디바운싱으로 오탐 방지 |
| **가시성** | 웹/앱을 통한 실시간 모니터링 |
| **확장성** | REST API 기반 유연한 연동 |

### 적용 분야

- 피트니스 센터
- 공공 체육시설
- 기업 복지시설
- 호텔/리조트 피트니스
